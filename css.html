CSS is a powerful tool for creating web pages. But there are many properties to keep straight. This page will act as a guide through the many systems and attributes that you can manipulate with CSS to make your web page stunning.

CSS Box Model

Padding Set: 20 pixels on all 4 sides. This is controlled by the padding property in CSS. and looks like padding: 20px; Entering a single number on this combined property sets all padding sides to the value.
Padding Set: 20 pixels on the top and bottom, and 50px on the left and right. The CSS property looks like padding: 20px 50px; Entering 2 values will have the first value control the top and bottom values, while the second one controls left and right values.
Padding Set: 100px 20px 40px 80px, Each of the 4 numbers correlates to a single side, starting from the top and then working around clockwise from there. Each of the 4 values corresponds to a single side.
Margin Set: none, This is controlled with the CSS property margin, though in this case we will only be using the left margin. Or margin-left. The default is also none.
Margin Set: 50px, as you can see the margin is the space around the box, and the box has been pushed over from the left side. This looks like: margin-left: 50px;
Margin Set: 140px, as you can see that increasing the margin value moved the margin much further. This looks like margin-left: 140px;

CSS Style on Hover

Before CSS, web pages were not responsive in design without adding code. However CSS offers limited responsive design with the hover hover property. This property allows you to have the appearance of something change when the mouse is over the item. This can simply have a color change, but you can potentially do a lot with this simple feature. To set an element on hover you simply need: element:hover in your css. This becomes especially powerful when you use transitions and transformations.

Hover Set: none, as you see nothing happens when you hover your mouse over.
Hover Set: Color, as you see, hovering your mouse over this slowly shifts the color. This is done by setting the hover with a different color, and using a transition: background-color, not in the same block as the hover.
Hover Set: Width, as you can see, hovering your mouse over this shifts the width of the box outwards. Similar to the color, you simply change the width of the element in a element:hover section in your css, and include a transition in the elements section.

CSS Shadows

Shadows are a powerful tool to creating parts of your page that really stand out. Shadows allow you to make text that looks almost 3D, though it can be used on more than just text. But we are just going to show you how you can use shadows on text. Shadows have 3 main properties which we can show you below. A horizontal value, a vertical value, and a blur.

Horizontal

As you can see below, horizontal value controls how far the shadow moves relative to the text. With larger numbers moving the text further and further from the actual text. You may also notice that a shadow is little more than a copy of the text, with blur giving it the actual appearance of being a shadow. But we will get to blur later. A negative value would move the shadow in the opposite direction. As seen by the 4th N7 below. It does work differently for box shadows as opposed to the text shadows. With box shadows This value does actually determine the size of the shadow.

Similar to Horizontal, Vertical simply controls the distance the shadow moves up or down from the actual text. And again you can see that with larger values, it moves the shadow further from the text. And negative values will instead move it in the opposite direction.

Blur

Instead of controlling the positioning of the shadow, blur instead determines amount of pixels that are taken to diffuse the shadow. Larger values allow the shadow to feather out more, creating those more gradual shadows as opposed to harder edged shadows. And as you can see, as the value gets larger the shadows take up a lot more room but end up looking more like a glow effect rather than an actual shadow. But that could potentially be the effect you are going for. Blur values cannot be negative, as it would not make sense to have negative space used for the blur effect. As you can see that it spreads the shadow out in all directions.
