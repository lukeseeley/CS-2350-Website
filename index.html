Favorite Games

Publishing Websites on Icarus

While it normally requires that you run your own web server or rent out a server to publish a web server, Weber States Computer Science department has made publishing websites on the internet a relative ease. We will go over how to publish websites through Icarus, a server that we can use to publish websites through our CS accounts. As well as the convention for URLs connecting back to these web pages that you add to the Icarus server.

Adding CSS Styles to Web Sites

CSS, along with HTML have really revolutionized how things work on the internet. CSS allows for powerful control over the appearance of web pages that just were not possible before without coding in Java, or PHP, among other languages. CSS is a very visual oriented way of designing web pages, and this serves as a guide to adding basic CSS styles to your web pages. Going over the various properties at your disposal to manipulating the page to appear the way you want it to.

Interactive Story: Lost in Space

A small but fun interactive mad lib that allows you to fill in the blanks to a peculiar story of a starship captain lost in space. Much like other mad libs you will first be asked to fill in several blanks of a variety of word types, such as adjectives, nouns, and verbs. Once you are ready you can hit submit and see your words fill the story.
